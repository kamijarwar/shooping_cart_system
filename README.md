# Shopping Cart System

## Description

A simple Python-based Shopping Cart System that allows users to add products to a cart and view all selected items at the end.

## Features

* Add multiple products
* Stop input using "done"
* Store items in a list
* Display total number of items
* Show all products in the cart
* Handle empty cart situations

## Technologies Used

* Python
* Lists
* Loops
* Conditional Statements

## Sample Output

Enter product name (or type 'done' to finish): Banana

Enter product name (or type 'done' to finish): Milk
